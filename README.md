# Honeypot
## 5. ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ
### 5.1. Требования к системе
* Операционная система: Linux

* Python версии 3.6 или выше

* Права доступа для создания сокетов (для портов выше 1024 достаточно прав пользователя)

### 5.2. Запуск honeypot
'''python
  Базовый запуск с портами по умолчанию
python3 honeypot_final_lab.py

  Запуск с указанием портов и файла логов
python3 honeypot_final_lab.py --ports 12222,18080,18443 --log honeypot.log

  Запуск с интервалом вывода статистики 60 секунд
python3 honeypot_final_lab.py --stats-interval 60
'''
### 5.3. Параметры командной строки
--host — хост для прослушивания (по умолчанию: 0.0.0.0)

--ports — список портов через запятую (по умолчанию: 12222,18080,18443)

--log — имя файла для логов (по умолчанию: lab4_honeypot.log)

--stats-interval — интервал вывода статистики в секундах (по умолчанию: 30)

### 5.4. Остановка работы
Для остановки honeypot необходимо нажать сочетание клавиш Ctrl+C в терминале, где запущена программа. Перед завершением будет выведена финальная статистика.
