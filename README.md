# Honeypot
## Инструкция по использованию
### Требования к системе
* Операционная система: Linux

* Python версии 3.6 или выше

* Права доступа для создания сокетов (для портов выше 1024 достаточно прав пользователя)

### Запуск honeypot
```ruby
  # Базовый запуск с портами по умолчанию
python3 honeypot_final_lab.py

  # Запуск с указанием портов и файла логов
python3 honeypot_final_lab.py --ports 2222,8080,8443 

  # Запуск с указанием портов и файла логов
python3 honeypot_final_lab.py --ports 2222,8080,8443 --log honeypot.log
```
### Параметры командной строки
* _--host_ — хост для прослушивания (по умолчанию: 0.0.0.0)

* _--ports_ — список портов через запятую (по умолчанию: 2222,8080,8443)

* _--log_ — имя файла для логов (по умолчанию: lab4_honeypot.log)

* _--stats-interval_ — интервал вывода статистики в секундах (по умолчанию: 30)

### Остановка работы
Для остановки honeypot необходимо нажать сочетание клавиш _Ctrl+C_ в терминале, где запущена программа. Перед завершением будет выведена финальная статистика.
